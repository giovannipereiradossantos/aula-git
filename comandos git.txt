comandos para iniciar git

1º cd\ - pasta raiz

2º md aula - cria pasta

3º cd aula - acessa a pasta

4º git init - inicilizar, novo repositorio, gerenciamneto de arquivo

5º git add* -  adicionar todos os arquivos

6º git commit - m "1 commit" nome que deseja
